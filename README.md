## Arquitectura del Proyecto
*   **`public/`**: Contiene los archivos estáticos (HTML, CSS, Imágenes).
*   **`routes/`**: Definición de los endpoints y rutas de la aplicación.
*   **`controllers/`**: Lógica de procesamiento de las peticiones.
*   **`middlewares/`**: Funciones de filtrado (Logs, validaciones de seguridad).
*   **`services/`**: Lógica de negocio pesada (opcional).
*   **`server.js`**: Punto de entrada y orquestador del servidor.

## Tecnologías Utilizadas
*   **Entorno:** Node.js
*   **Framework:** Express.js
*   **Seguridad:** Cookie-parser (Gestión de sesiones)
*   **Maquetación:** HTML5 Semántico y CSS Flexbox

## Instalación y Ejecución
Para levantar este proyecto en un ambiente local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone [URL-DEL-REPOSITORIO]
    ```
2.  **Inicializar el proyecto e instalar dependencias:**
    ```bash
    npm install
    ```
    *(Asegúrate de que el archivo `package.json` incluya `express` y `cookie-parser`)* [5].
3.  **Iniciar el servidor:**
    ```bash
    node server.js
    ```
4.  **Acceso:**
    Abre tu navegador en `http://localhost:3000`
